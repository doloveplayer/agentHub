# 消息轮次操作（Turn-Based Message Operations）实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 引入 ConversationTurn 数据模型，将消息删除/重新生成操作从单条消息提升到对话轮次级别，同时支持单 Agent 撤回（软删除+占位符），并正确处理产物（sandbox 文件、ContextEntry、PlanExecution）的撤销与保留。

**Architecture:** 新增 `ConversationTurn` Prisma 模型作为 Message 的父级分组单元。一个 Turn = 一次用户意图触发的完整多 agent 协作周期。Turn 级删除级联清理所有关联消息和产物，Turn 级重新生成创建新 Turn 并保留旧版本用于切换。利用现有 `WorkspaceManager.snapshot/rollback` 实现工作区回滚，利用 `diffBroadcast` 的版本跟踪机制做 Turn 边界标记。

**Tech Stack:** Prisma + PostgreSQL (schema migration), Hono REST API, WebSocket (ws/handler.ts), React + Zustand, 现有 WorkspaceManager (git stash/branch snapshot)

## Task 3: TurnManager — 核心生命周期管理

**Files:**
- Create: `apps/api/src/agent/TurnManager.ts`

- [ ] **Step 1: 创建 TurnManager 类**

```typescript
import { prisma } from '../db.js';
import { WorkspaceManager, type Snapshot } from './WorkspaceManager.js';
import type { ConversationTurn, WorkspaceSnapshot } from '@agenthub/shared';

const TURN_MESSAGES_LIMIT = 100; // Turn 最多包含的消息数（防止无限增长）

export class TurnManager {
  /**
   * 为新用户消息创建 Turn（或复用当前活跃 Turn）。
   * 规则：
   * - 如果当前有 active Turn，先 complete 它
   * - 创建新 Turn，sequence = 当前 session 最大 sequence + 1
   * - 创建 workspace snapshot（git stash）用于后续回滚
   * - 返回新 Turn
   */
  static async createTurn(params: {
    sessionId: string;
    userMessageId: string;
    workspacePath?: string | null;
  }): Promise<ConversationTurn> {
    const { sessionId, userMessageId, workspacePath } = params;

    // 1. 完成当前活跃 Turn
    await prisma.conversationTurn.updateMany({
      where: { sessionId, status: 'active' },
      data: { status: 'completed' },
    });

    // 2. 计算 sequence
    const lastTurn = await prisma.conversationTurn.findFirst({
      where: { sessionId },
      orderBy: { sequence: 'desc' },
      select: { sequence: true },
    });
    const sequence = (lastTurn?.sequence ?? 0) + 1;

    // 3. Workspace snapshot（如果提供了 workspacePath）
    let workspaceSnapshot: WorkspaceSnapshot | null = null;
    if (workspacePath) {
      const snapshot = WorkspaceManager.snapshot(workspacePath, sessionId);
      if (snapshot) {
        workspaceSnapshot = {
          ref: snapshot.ref,
          mode: snapshot.mode,
          workspacePath: snapshot.workspacePath,
          createdAt: snapshot.createdAt,
        };
      }
    }

    // 4. 创建 Turn
    const turn = await prisma.conversationTurn.create({
      data: {
        sessionId,
        sequence,
        triggerMsgId: userMessageId,
        status: 'active',
        workspaceSnapshot: workspaceSnapshot as any,
        contextEntryKeys: [],
        planIds: [],
      },
    });

    // 5. 关联用户消息到 Turn
    await prisma.message.update({
      where: { id: userMessageId },
      data: { turnId: turn.id },
    });

    return turn as unknown as ConversationTurn;
  }

  /**
   * Agent 消息创建时关联到当前 active Turn
   */
  static async associateMessage(turnId: string, messageId: string): Promise<void> {
    await prisma.message.update({
      where: { id: messageId },
      data: { turnId },
    });
  }

  /**
   * 获取或创建当前 active Turn
   */
  static async getActiveTurn(sessionId: string): Promise<ConversationTurn | null> {
    const turn = await prisma.conversationTurn.findFirst({
      where: { sessionId, status: 'active' },
      orderBy: { sequence: 'desc' },
    });
    return turn as unknown as ConversationTurn | null;
  }

  /**
   * 记录 Turn 期间创建的 ContextEntry key
   */
  static async trackContextEntry(turnId: string, key: string): Promise<void> {
    const turn = await prisma.conversationTurn.findUnique({ where: { id: turnId }, select: { contextEntryKeys: true } });
    if (!turn) return;
    const keys = [...new Set([...(turn.contextEntryKeys as string[]), key])];
    await prisma.conversationTurn.update({ where: { id: turnId }, data: { contextEntryKeys: keys } });
  }

  /**
   * 记录 Turn 期间创建的 PlanExecution planId
   */
  static async trackPlanId(turnId: string, planId: string): Promise<void> {
    const turn = await prisma.conversationTurn.findUnique({ where: { id: turnId }, select: { planIds: true } });
    if (!turn) return;
    const planIds = [...new Set([...(turn.planIds as string[]), planId])];
    await prisma.conversationTurn.update({ where: { id: turnId }, data: { planIds } });
  }

  /**
   * 完成 Turn（标记 completed，断开 plan 关联）
   */
  static async completeTurn(turnId: string): Promise<void> {
    await prisma.conversationTurn.update({
      where: { id: turnId },
      data: { status: 'completed' },
    });
  }

  /**
   * 删除 Turn：
   * 1. 回滚 workspace 到 turn 前的 snapshot
   * 2. 标记 ContextEntry 为 stale
   * 3. 取消 PlanExecution
   * 4. 清理 QuoteReference（涉及该 Turn 消息的引用）
   * 5. 清理 PinnedMessage（来源是该 Turn 消息的 pin）
   * 6. 删除所有关联消息（除了标记为 undone 的可选保留）
   * 7. 删除 Turn 记录
   *
   * @param options.keepUndoneAsPlaceholder 是否保留 undone 消息作为占位符（默认 false = 全部删除）
   */
  static async deleteTurn(
    turnId: string,
    options: { keepUndoneAsPlaceholder?: boolean } = {},
  ): Promise<{
    revertedFiles: string[];
    deletedMessageIds: string[];
    undoneMessageIds: string[];
  }> {
    const turn = await prisma.conversationTurn.findUnique({
      where: { id: turnId },
      include: { messages: { select: { id: true, senderType: true, agentId: true, turnStatus: true } } },
    });
    if (!turn) throw new Error(`Turn ${turnId} not found`);

    // 1. Workspace 回滚
    const revertedFiles: string[] = [];
    const snapshot = turn.workspaceSnapshot as WorkspaceSnapshot | null;
    if (snapshot) {
      const success = WorkspaceManager.rollback({
        ref: snapshot.ref,
        mode: snapshot.mode,
        workspacePath: snapshot.workspacePath,
        createdAt: snapshot.createdAt,
      });
      if (success) {
        revertedFiles.push(...WorkspaceManager.getChanges(snapshot.workspacePath));
      }
    }

    // 2. ContextEntry 标记为 stale
    const contextKeys = turn.contextEntryKeys as string[];
    if (contextKeys.length > 0) {
      await prisma.contextEntryRecord.updateMany({
        where: { key: { in: contextKeys }, sessionId: turn.sessionId },
        data: { status: 'stale' },
      });
    }

    // 3. 取消 PlanExecution
    const planIds = turn.planIds as string[];
    if (planIds.length > 0) {
      await prisma.planExecution.updateMany({
        where: { planId: { in: planIds }, sessionId: turn.sessionId },
        data: { status: 'failed' },
      });
    }

    // 4. 清理 QuoteReference
    const allMessageIds = turn.messages.map((m) => m.id);
    await prisma.quoteReference.deleteMany({
      where: {
        sessionId: turn.sessionId,
        OR: [
          { sourceMessageId: { in: allMessageIds } },
          { targetMessageId: { in: allMessageIds } },
        ],
      },
    });

    // 5. 清理 PinnedMessage（来源是该 Turn 消息的 pin）
    await prisma.pinnedMessage.deleteMany({
      where: {
        sessionId: turn.sessionId,
        sourceMessageId: { in: allMessageIds },
      },
    });

    // 6. 删除消息
    const { keepUndoneAsPlaceholder } = options;
    let deletedIds: string[] = [];
    let undoneIds: string[] = [];

    if (keepUndoneAsPlaceholder) {
      // 软删除：将 undone 消息标记但保留
      const undoneMessages = turn.messages.filter((m) => m.turnStatus === 'undone');
      const regularMessages = turn.messages.filter((m) => m.turnStatus !== 'undone');
      undoneIds = undoneMessages.map((m) => m.id);
      deletedIds = regularMessages.map((m) => m.id);

      if (deletedIds.length > 0) {
        await prisma.message.deleteMany({ where: { id: { in: deletedIds } } });
      }
      // undone 消息保留但内容设为占位符
      if (undoneIds.length > 0) {
        await prisma.message.updateMany({
          where: { id: { in: undoneIds } },
          data: { content: '', turnStatus: 'undone' },
        });
      }
    } else {
      deletedIds = allMessageIds;
      await prisma.message.deleteMany({ where: { id: { in: deletedIds } } });
    }

    // 7. 标记 Turn 为 cancelled
    await prisma.conversationTurn.update({
      where: { id: turnId },
      data: { status: 'cancelled' },
    });

    return { revertedFiles, deletedMessageIds: deletedIds, undoneMessageIds: undoneIds };
  }

  /**
   * 重新生成 Turn：创建新 Turn（parentTurnId = 旧 Turn），
   * 发送修改后的用户 prompt，旧 Turn 标记为 superseded。
   *
   * 策略：回滚 workspace 到旧 Turn 快照 → 创建新 Turn → 重新调度
   */
  static async regenerateTurn(
    oldTurnId: string,
    editContent?: string,
  ): Promise<{
    oldTurnId: string;
    newTurnId: string;
    newSequence: number;
    originalContent: string;   // 原始用户消息内容（或编辑后的）
  }> {
    const oldTurn = await prisma.conversationTurn.findUnique({
      where: { id: oldTurnId },
      include: {
        messages: {
          where: { senderType: 'human' },
          orderBy: { createdAt: 'asc' },
          take: 1,
          select: { content: true },
        },
      },
    });
    if (!oldTurn) throw new Error(`Turn ${oldTurnId} not found`);

    const originalContent = editContent || oldTurn.messages[0]?.content || '';

    // 1. 回滚 workspace 到旧 Turn 快照
    const snapshot = oldTurn.workspaceSnapshot as WorkspaceSnapshot | null;
    if (snapshot) {
      WorkspaceManager.rollback({
        ref: snapshot.ref,
        mode: snapshot.mode,
        workspacePath: snapshot.workspacePath,
        createdAt: snapshot.createdAt,
      });
    }

    // 2. 标记旧 Turn 为 superseded
    await prisma.conversationTurn.update({
      where: { id: oldTurnId },
      data: { status: 'superseded' },
    });

    // 3. 计算新 sequence
    const lastTurn = await prisma.conversationTurn.findFirst({
      where: { sessionId: oldTurn.sessionId },
      orderBy: { sequence: 'desc' },
      select: { sequence: true },
    });
    const newSequence = (lastTurn?.sequence ?? 0) + 1;

    // 4. 创建新 Turn（继承 workspaceSnapshot 但 ref 已过期，需重新 snapshot）
    const newSnapshot = snapshot
      ? WorkspaceManager.snapshot(snapshot.workspacePath, oldTurn.sessionId)
      : null;
    const newWorkspaceSnapshot = newSnapshot
      ? { ref: newSnapshot.ref, mode: newSnapshot.mode, workspacePath: newSnapshot.workspacePath, createdAt: newSnapshot.createdAt }
      : null;

    const newTurn = await prisma.conversationTurn.create({
      data: {
        sessionId: oldTurn.sessionId,
        sequence: newSequence,
        parentTurnId: oldTurnId,
        triggerMsgId: '', // 由 chat route 创建消息后回填
        status: 'active',
        workspaceSnapshot: newWorkspaceSnapshot as any,
        contextEntryKeys: [],
        planIds: [],
      },
    });

    return {
      oldTurnId,
      newTurnId: newTurn.id,
      newSequence: newTurn.sequence,
      originalContent,
    };
  }

  /**
   * 单 Agent 消息撤回（软删除）：
   * 标记消息 turnStatus = 'undone'，内容清空
   * 不触发 workspace 回滚，不影响其他 Agent 消息
   */
  static async undoAgentMessage(messageId: string): Promise<{
    messageId: string;
    agentId: string;
    turnId: string;
  }> {
    const message = await prisma.message.findUnique({
      where: { id: messageId },
      select: { id: true, agentId: true, turnId: true, senderType: true },
    });
    if (!message || message.senderType !== 'agent') {
      throw new Error(`Agent message ${messageId} not found`);
    }

    await prisma.message.update({
      where: { id: messageId },
      data: {
        content: '',
        turnStatus: 'undone',
      },
    });

    return {
      messageId: message.id,
      agentId: message.agentId || '',
      turnId: message.turnId || '',
    };
  }

  /**
   * 获取 Turn 版本链（parentTurnId 链）
   */
  static async getTurnVersions(rootTurnId: string): Promise<ConversationTurn[]> {
    // 先找到根 Turn (最远的祖先)
    let current = await prisma.conversationTurn.findUnique({ where: { id: rootTurnId } });
    if (!current) return [];

    // 向上找到根
    while (current!.parentTurnId) {
      const parent = await prisma.conversationTurn.findUnique({
        where: { id: current!.parentTurnId },
      });
      if (!parent) break;
      current = parent;
    }

    // 从根向下收集所有版本
    const versions: ConversationTurn[] = [current as unknown as ConversationTurn];
    const children = await prisma.conversationTurn.findMany({
      where: { parentTurnId: current!.id },
      orderBy: { sequence: 'asc' },
    });
    for (const child of children) {
      versions.push(child as unknown as ConversationTurn);
      // 递归子节点（实际深度通常为 1-2 层）
      const grandchildren = await prisma.conversationTurn.findMany({
        where: { parentTurnId: child.id },
        orderBy: { sequence: 'asc' },
      });
      for (const gc of grandchildren) {
        versions.push(gc as unknown as ConversationTurn);
      }
    }

    return versions;
  }
```

```typescript
import { ChevronLeft, ChevronRight } from 'lucide-react';
import { useState, useEffect } from 'react';
import { api } from '../lib/api';
import type { TurnVersion } from '@agenthub/shared';

interface VersionSwitcherProps {
  turnId: string;
  currentVersionTurnId: string;
  onSwitchVersion: (version: TurnVersion) => void;
}

export function VersionSwitcher({ turnId, currentVersionTurnId, onSwitchVersion }: VersionSwitcherProps) {
  const [versions, setVersions] = useState<TurnVersion[]>([]);
  const [currentIndex, setCurrentIndex] = useState(0);

  useEffect(() => {
    api.getTurnVersions(turnId).then((res) => {
      setVersions(res.versions);
      const idx = res.versions.findIndex((v) => v.turnId === currentVersionTurnId);
      if (idx >= 0) setCurrentIndex(idx);
    });
  }, [turnId, currentVersionTurnId]);

  if (versions.length <= 1) return null;

  return (
    <div className="flex items-center gap-2 px-3 py-1 rounded-full bg-hub-surface border border-hub-border text-xs">
      <button
        onClick={() => {
          const newIdx = Math.max(0, currentIndex - 1);
          setCurrentIndex(newIdx);
          onSwitchVersion(versions[newIdx]);
        }}
        disabled={currentIndex === 0}
        className="p-0.5 hover:text-hub-accent disabled:opacity-30"
      >
        <ChevronLeft className="w-3 h-3" />
      </button>
      <span className="text-hub-muted">
        {currentIndex + 1} / {versions.length}
      </span>
      <button
        onClick={() => {
          const newIdx = Math.min(versions.length - 1, currentIndex + 1);
          setCurrentIndex(newIdx);
          onSwitchVersion(versions[newIdx]);
        }}
        disabled={currentIndex === versions.length - 1}
        className="p-0.5 hover:text-hub-accent disabled:opacity-30"
      >
        <ChevronRight className="w-3 h-3" />
      </button>
    </div>
  );
}
```

## 产物撤销与保留策略总结

| 产物类型 | Turn 删除 | Turn 重新生成 | 单 Agent 撤回 |
|---------|-----------|-------------|-------------|
| **沙箱文件** (workspace) | git stash 回滚到 Turn 前状态 | 回滚到旧 Turn 快照 → 重新执行 | 不处理（其他 Agent 文件保留） |
| **ContextEntry** | 标记为 `stale` | 旧 Turn 的标记 stale，新 Turn 重新创建 | 不处理 |
| **PlanExecution** | 标记为 `failed` | 旧 Plan 标记 failed，新 Turn 创建新 Plan | 不处理（Plan 是 Turn 级的） |
| **QuoteReference** | 涉及消息的引用全部删除 | 旧引用删除，新 Turn 创建新引用 | 仅该消息的引用清理 |
| **PinnedMessage** | 来源为该 Turn 消息的 Pin 删除 | 旧 Pin 删除 | 不处理 |
| **Message** | 物理删除（可选保留 undone 占位符） | 旧 Turn 消息保留（status=superseded），创建新消息 | 软删除（content 清空，turnStatus=undone） |
